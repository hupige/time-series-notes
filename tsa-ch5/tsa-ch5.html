5.1 Cramer分解定理

在讨论非平稳序列的分析方法之前，我们需要首先理解非平稳序列的分解结构。Cramer分解定理为时间序列的随机分析提供了理论基础。

Cramer分解定理（回顾） 任何一个时间序列 $\{x_t\}$ 都可以分解为确定性成分与随机成分之和： $$x_t = D_t + S_t$$ 其中 $D_t$ 为确定性趋势成分（可用多项式、季节函数等描述），$S_t$ 为平稳的随机成分。

基于Cramer分解，非平稳序列的处理存在两条路径：

确定性分析

提取确定性趋势 $D_t$（如回归、移动平均趋势拟合），对残差 $\hat{S}_t = x_t - \hat{D}_t$ 建立ARMA模型。

随机分析（本章重点）

通过差分运算消除确定性趋势，将原序列转化为平稳序列，再对差分后序列建立ARMA模型。

差分运算的本质，正是利用相邻观测值之间的变化来消除确定性趋势，使序列转化为平稳的随机成分。与确定性分析相比，随机分析不需要预先设定趋势函数的具体形式，具有更强的灵活性和适应性。

5.2 差分运算

15.2.1 差分运算的实质

差分（Differencing）是一种线性滤波运算，其核心思想是提取相邻观测值之间的变化量。当序列中存在确定性趋势时，差分运算可以有效地消除这种趋势。

一阶差分（First-order differencing） $$\nabla x_t = x_t - x_{t-1}$$
二阶差分（Second-order differencing） $$\nabla^2 x_t = \nabla x_t - \nabla x_{t-1} = x_t - 2x_{t-1} + x_{t-2}$$
p步差分（Lag-p differencing） $$\nabla_p x_t = x_t - x_{t-p}$$

其中，$\nabla$ 称为差分算子（difference operator）。一阶差分消除线性趋势，二阶差分消除二次曲线趋势，$p$ 步差分则用于消除周期为 $p$ 的季节性成分。

25.2.2 差分方式的选择

选择适当的差分方式是非平稳序列建模的关键第一步。不同类型的非平稳性对应不同的差分策略：

趋势类型序列特征推荐差分方式差分后序列
线性趋势 $x_t = a + bt + \varepsilon_t$ 一阶差分 $\nabla x_t$ 平稳序列（均值为 $b$）
曲线趋势 $x_t = a + bt + ct^2 + \varepsilon_t$ 二阶差分 $\nabla^2 x_t$ 平稳序列
固定周期趋势 周期为 $s$ 的季节波动 $s$ 步差分 $\nabla_s x_t$ 消除季节性后的平稳序列
实践建议 在实际建模中，通常从最低阶差分开始尝试。若一阶差分后序列已平稳，则无需更高阶差分。可借助ADF检验（Augmented Dickey-Fuller Test）或KPSS检验来判断序列是否达到平稳。

35.2.3 过差分

过差分（Over-differencing）是指对序列进行了不必要的、过度的差分操作。虽然过度差分后的序列在理论上仍然平稳，但会导致模型结构复杂化、参数估计效率降低。

过差分的危害 以随机游走（Random Walk）模型为例：$x_t = x_{t-1} + \varepsilon_t$，其中 $\varepsilon_t \sim WN(0,\sigma^2)$。该序列只需一阶差分即可得到白噪声：$\nabla x_t = \varepsilon_t$。若进行二阶差分：$\nabla^2 x_t = \varepsilon_t - \varepsilon_{t-1}$，这实际上是一个MA(1)模型，方差增大且引入了不必要的可逆性约束，造成了信息浪费。
金融案例：股价对数收益率 在金融工程中，股票价格 $P_t$ 通常呈现非平稳的随机游走特征。对数收益率定义为： $$r_t = \ln P_t - \ln P_{t-1} = \nabla \ln P_t$$ 这恰好是对数股价的一阶差分！因此，收益率序列 $\{r_t\}$ 的平稳性分析本质上就是差分运算在金融领域的最直接应用。

5.3 ARIMA模型

15.3.1 ARIMA(p,d,q)的结构

ARIMA（Autoregressive Integrated Moving Average）模型是ARMA模型向非平稳序列的自然推广。其基本思想是：对原序列进行 $d$ 阶差分后，对差分后序列建立ARMA(p,q)模型。

ARIMA(p,d,q)模型的一般形式 $$\nabla^d x_t = \varphi_0 + \sum_{i=1}^{p} \varphi_i \nabla^d x_{t-i} + \varepsilon_t - \sum_{j=1}^{q} \theta_j \varepsilon_{t-j}$$

其中参数含义如下：

$p$ — 自回归阶数

差分后序列的自回归项数，决定历史值对当前值的影响深度。

$d$ — 差分阶数

使序列平稳所需的最小差分次数，反映序列的"单整"阶数。

$q$ — 移动平均阶数

差分后序列的移动平均项数，捕捉残差中的短期依赖。

ARIMA模型包含两个重要的退化情形：

  • 当 $d=0$ 时，ARIMA(p,0,q) 退化为普通的 ARMA(p,q) 模型；
  • 当 $d=1, p=q=0$ 时，ARIMA(0,1,0) 即为随机游走模型。

25.3.2 ARIMA的性质

理解ARIMA模型的性质对于正确应用至关重要。以下是其核心性质：

差分后序列平稳

经过 $d$ 阶差分后，序列 $\{\nabla^d x_t\}$ 为平稳序列，满足ARMA模型的所有性质（均值恒定、方差有限、自协方差仅依赖于滞后阶数）。

原序列非平稳

当 $d \geq 1$ 时，原序列 $\{x_t\}$ 的方差随时间发散，自协方差依赖于时间起点而非仅依赖滞后阶数。

ARIMA(0,1,0) = 随机游走

$$x_t = x_{t-1} + \varepsilon_t$$ 是金融中最基础也最重要的非平稳模型，股价建模的经典起点。

冲击的持久性

在ARIMA模型中，随机冲击 $\varepsilon_t$ 对序列的影响不会随时间衰减（当 $d \geq 1$ 时），体现出"记忆持久"的特征。

35.3.3 ARIMA建模步骤

ARIMA模型的建模是一个系统化的流程，遵循"平稳化—识别—估计—检验—优化—预测"的六步框架：

  1. 平稳化（差分至平稳） 对原始序列进行ADF检验或观察时序图、ACF图。若序列非平稳，进行差分运算直至平稳，确定差分阶数 $d$。
  2. 模型识别（ACF/PACF定阶） 对平稳的差分后序列，观察样本ACF和PACF的截尾/拖尾特征，初步确定 $p$ 和 $q$ 的阶数范围。
  3. 参数估计（MLE / LS） 采用极大似然估计（MLE）或最小二乘估计（LS）对模型参数进行估计，得到 $\hat{\varphi}_i$ 和 $\hat{\theta}_j$。
  4. 模型检验（残差白噪声检验） 检验残差序列是否为白噪声。常用Ljung-Box检验：若残差非白噪声，说明模型提取信息不充分，需重新定阶。
  5. 模型优化（AIC / BIC准则） 在多个候选模型中，根据AIC（Akaike Information Criterion）或BIC（Bayesian Information Criterion）选择最优模型，平衡拟合优度与模型复杂度。
  6. 预测 利用选定模型进行未来值的预测，并给出预测区间。ARIMA预测具有最小均方误差性质。

45.3.4 ARIMA预测

ARIMA模型的预测基于最小均方误差预测（Minimum Mean Square Error Forecast）原理。对于ARIMA(p,d,q)模型，$l$ 步向前预测 $\hat{x}_{T}(l)$ 是使预测误差方差 $E[(x_{T+l} - \hat{x}_{T}(l))^2]$ 达到最小的预测值。

最小均方误差预测原理 $$\hat{x}_{T}(l) = E[x_{T+l} \mid x_T, x_{T-1}, \dots]$$ 即未来值的条件期望即为最优预测。预测误差方差随预测步长 $l$ 增大而增大，反映不确定性累积。

对于ARIMA模型，预测具有如下特点：

  • 短期预测精度较高，长期预测趋于序列的均值水平；
  • 预测区间（置信区间）随预测步长扩大而增宽；
  • 对于 $d \geq 1$ 的模型，长期预测值最终收敛于一条确定性趋势直线。
金融经典

ARIMA(0,1,0) 随机游走模型

$$x_t = x_{t-1} + \varepsilon_t, \quad \varepsilon_t \sim WN(0,\sigma^2)$$

金融意义：随机游走是金融中股价建模的基石。在有效市场假说下，资产价格的变动完全不可预测，下一期价格等于当期价格加上随机冲击。对数收益率 $r_t = \nabla \ln P_t$ 在此模型下为白噪声。

统计性质：$Var(x_t) = t\sigma^2$，方差随时间线性增长；$Cov(x_t, x_{t-k}) = (t-k)\sigma^2$；自相关系数 $\rho_k \approx \sqrt{(t-k)/t} \to 1$（当 $t$ 很大时）。

常用模型

ARIMA(1,1,0) 与 ARIMA(0,1,1)

ARIMA(1,1,0): $\nabla x_t = \varphi_1 \nabla x_{t-1} + \varepsilon_t$   |   ARIMA(0,1,1): $\nabla x_t = \varepsilon_t - \theta_1 \varepsilon_{t-1}$

ARIMA(1,1,0)：差分后序列服从AR(1)过程，当前变化率受上一期变化率的影响。适用于具有惯性特征的经济序列（如利率）。

ARIMA(0,1,1)：差分后序列服从MA(1)过程，当前变化受当期和上一期冲击的共同影响。等价于指数平滑模型，在库存管理和需求预测中广泛应用。

5.4 疏系数模型

在实际应用中，ARIMA模型的某些中间阶数参数可能为零，此时模型称为疏系数模型（Sparse Coefficient Model）或季节性ARIMA的特殊形式。

疏系数表示法 当自回归或移动平均多项式中仅存在部分非零系数时，采用括号标记法。例如：

ARIMA((1,3),1,0) 表示对一阶差分后的序列建立AR模型，其中仅包含1阶和3阶自回归项，2阶自回归系数为零： $$\nabla x_t = \varphi_1 \nabla x_{t-1} + \varphi_3 \nabla x_{t-3} + \varepsilon_t$$

疏系数模型的出现通常源于数据本身的周期性结构或经济机制的间隔性影响。在建模过程中，若某些阶数的参数估计值不显著（t检验未通过），可以考虑将其约束为零，从而得到更简洁、更可解释的疏系数模型。

本章系统介绍了非平稳时间序列的随机分析方法，核心框架可概括为以下流程：

ARIMA建模全流程
原始非平稳序列 $\{x_t\}$
→
差分平稳化 $\nabla^d x_t$

→
残差检验
→
AIC/BIC选优
→

与第4章的确定性分析方法相比，随机分析具有以下特点：

对比维度确定性分析随机分析（ARIMA）
趋势处理 显式拟合趋势函数（多项式/季节函数） 差分运算隐式消除趋势，无需预设函数形式
模型假设 假设趋势结构已知 通过数据驱动确定差分阶数
灵活性 对复杂趋势的适应性有限 对多种非平稳类型具有统一处理框架
预测性质 外推确定性趋势 最小均方误差预测，含预测区间
金融适用性 难以捕捉市场有效性的随机特征 随机游走是有效市场假说的数学表达
核心要点回顾 差分运算是非平稳序列随机分析的基石；ARIMA(p,d,q)通过"差分+ARMA"的统一框架为各类非平稳序列提供建模工具；在金融工程中，ARIMA(0,1,0)随机游走模型是资产定价、风险管理和衍生品定价的理论基础。掌握差分的"度"——既不过度也不过少——是ARIMA建模的艺术所在。
