Chapter 03

平稳时间序列分析

深入理解ARMA模型的理论基础与统计性质，掌握自回归、移动平均及混合模型的识别方法与金融应用。

ARMA模型 ACF / PACF 平稳性 & 可逆性

3.1 Wold分解定理

Wold分解定理（Wold Decomposition Theorem）是时间序列分析的理论基石，它揭示了任何离散平稳时间序列的内在结构。

Wold分解定理

任何离散平稳时间序列都可以唯一地分解为确定性成分和随机成分之和：

式 3-1 xt = Vt + ηt

其中：

  • Vt 为确定性序列（Deterministic Component），可被历史值完全预测
  • ηt 为随机序列（Stochastic Component），是无限阶移动平均过程

随机成分 ηt 可进一步表示为：

式 3-2 ηt = ∑j=0∞ ψj εt-j

其中 ψ0 = 1，∑j=0∞ ψj2 < ∞，εt 为白噪声过程。

理论意义

Wold分解定理表明：对于平稳序列，只要建立适当的ARMA模型，就能充分捕捉其动态特征。这为ARMA模型提供了坚实的理论基础——确定性部分可用AR结构逼近，随机部分可用MA结构刻画。

3.2 AR模型（自回归模型）

3.2.1 AR(p)定义

自回归模型（AutoRegressive Model, AR）的核心思想是：当前时刻的观测值可以由其历史值的线性组合加上随机扰动来解释。

式 3-3 xt = φ0 + φ1 xt-1 + φ2 xt-2 + … + φp xt-p + εt

引入滞后算子 L（其中 Lk xt = xt-k），AR(p)模型可简洁表示为：

式 3-4 Φ(L) xt = φ0 + εt

其中 Φ(L) = 1 - φ1L - φ2L2 - … - φpLp 称为自回归特征多项式。

3.2.2 平稳性判别

并非所有AR(p)过程都是平稳的。平稳性判别有以下两种等价方法：

特征根法
求解特征方程 Φ(λ) = 0 的所有根。若所有特征根的模都在单位圆内（即 |λi| < 1），则AR(p)过程平稳。
平稳域法
通过模型参数 φ1, …, φp 构成的约束条件判断。例如AR(1)的平稳域为 |φ1| < 1；AR(2)的平稳域为三角形区域。
注意区分

有些教材使用"特征方程 1 - φ1z - … - φpzp = 0 的根在单位圆外"作为平稳条件——这与"特征根在单位圆内"是等价的，只是特征多项式定义方式不同，切勿混淆。

3.2.3 统计性质

对于零均值的平稳AR(p)过程，其统计性质可推导如下：

均值 E(xt) = μ = φ0 / (1 - φ1 - … - φp)
方差 Var(xt) = γ0 = σε2 / (1 - φ1ρ1 - … - φpρp)
自协方差函数 γk = φ1γk-1 + φ2γk-2 + … + φpγk-p 　(k ≥ 1)

上述自协方差递推关系称为Yule-Walker方程，是AR模型参数估计的重要工具。

3.2.4 自相关系数(ACF)：拖尾特征

AR(p)过程的自相关系数(ACF)具有拖尾（Tail-off）特征：即 ρk 不会在某阶后突然截断为零，而是呈指数衰减或正弦振荡衰减。

式 3-5 ρk = φ1ρk-1 + φ2ρk-2 + … + φpρk-p 　(k ≥ 1)

这一递推关系说明：AR(p)的ACF被其特征方程的根所控制，尾部被负指数函数控制而趋于零。

3.2.5 偏自相关系数(PACF)：p阶截尾特征

偏自相关系数（Partial Autocorrelation Function, PACF）衡量了剔除中间滞后项影响后，xt 与 xt-k 的净相关关系。

AR(p)的PACF核心性质

AR(p)过程的偏自相关系数具有p阶截尾特征：

φkk = 0，对于所有 k > p

这意味着，若序列的PACF在滞后p阶后突然"截断"为零，则提示该序列可能服从AR(p)模型。

AR(1)
一阶自回归模型

平稳条件
|φ1| < 1

AR(1)过程具有均值回归（Mean Reversion）特性：当 φ1 > 0 时，偏离均值的冲击会逐渐衰减回复到长期均值。利率、汇率等金融变量常呈现此特征。

3.3 MA模型（移动平均模型）

3.3.1 MA(q)定义

移动平均模型（Moving Average Model, MA）描述的是当前观测值受当前及历史随机冲击影响的结构。

式 3-6 xt = μ + εt - θ1 εt-1 - θ2 εt-2 - … - θq εt-q

使用滞后算子表示为：

式 3-7 xt = μ + Θ(L) εt

其中 Θ(L) = 1 - θ1L - θ2L2 - … - θqLq 称为移动平均特征多项式。

3.3.2 统计性质：有限记忆性

MA(q)过程具有有限记忆性（Finite Memory）：当前观测值仅受最近q期的随机冲击影响。

均值 E(xt) = μ
方差 Var(xt) = σε2 (1 + θ12 + θ22 + … + θq2)

3.3.3 ACF：q阶截尾

MA(q)过程的自协方差函数在滞后q阶后严格为零：

式 3-8 γk = 0 　(k > q)

因此，MA(q)的自相关系数(ACF)具有q阶截尾特征。这是识别MA模型阶数的关键依据。

3.3.4 PACF：拖尾

与AR模型相反，MA(q)过程的偏自相关系数(PACF)呈现拖尾特征，被负指数函数控制而逐渐衰减至零。

MA(1)
一阶移动平均模型
xt = μ + εt - θ1 εt-1
可逆条件
|θ1| < 1
方差
γ0 = σε2(1 + θ12)
ACF
ρ1 = -θ1/(1+θ12)；ρk = 0 (k > 1) （1阶截尾）
PACF
拖尾衰减
金融含义
MA(1)模型刻画了市场冲击的短期影响：一个意外冲击 εt 仅影响当期和下一期观测值，之后完全消散。适用于描述收益率对新闻事件的短暂反应。
MA(2)
二阶移动平均模型

MA(2)适用于市场冲击效应持续两期的场景，例如某些宏观经济 announcements 对市场情绪的影响可能持续多个交易日。

3.4 ARMA模型

3.4.1 ARMA(p,q)定义

自回归移动平均模型（AutoRegressive Moving Average Model, ARMA）将AR和MA结构相结合，以更灵活地刻画时间序列的动态特征。

式 3-9 xt = φ0 + φ1 xt-1 + … + φp xt-p + εt - θ1 εt-1 - … - θq εt-q

使用滞后算子表示：

式 3-10 Φ(L) xt = φ0 + Θ(L) εt

ARMA(p,q)模型兼具了AR的"记忆性"和MA的"冲击响应"特征，是实践中应用最广泛的平稳时间序列模型。

3.4.2 平稳性与可逆性条件

ARMA(p,q)过程需要同时满足两个条件：

平稳性条件（AR部分）
AR特征方程 Φ(λ) = 0 的所有根在单位圆内（或等价的 Φ(z) = 0 根在单位圆外）。保证序列的均值、方差和自协方差不随时间变化。
可逆性条件（MA部分）
MA特征方程 Θ(λ) = 0 的所有根在单位圆内（或等价的 Θ(z) = 0 根在单位圆外）。保证MA过程可表示为收敛的无穷阶AR过程。
为什么要可逆性？

可逆性保证了MA(q)过程可以唯一地由当前和过去的观测值表示（即存在唯一的AR(∞)展开）。这确保了模型参数的可识别性，使得我们可以根据观测数据唯一估计出模型参数。

3.4.3 ACF和PACF：都是拖尾

ARMA(p,q)过程的ACF和PACF具有一个关键特征：两者都是拖尾的。

  • AR(p)的ACF拖尾，PACF p阶截尾
  • MA(q)的ACF q阶截尾，PACF拖尾
  • ARMA(p,q)的ACF和PACF都拖尾（尾部被负指数控制）

这意味着，当观测到ACF和PACF都呈现拖尾特征时，提示需要使用ARMA混合模型，而非单纯的AR或MA模型。

金融案例：ARMA在资产收益率建模中的应用

资产收益率序列通常表现出以下特征，使得ARMA模型成为自然的选择：

  1. 微弱自相关：收益率本身往往存在微弱的序列相关性，可用低阶AR项捕捉
  2. 冲击效应：市场新闻和事件对收益率的冲击具有持续性，可用MA项描述
  3. 混合动态：许多金融序列同时表现出AR和MA特征，ARMA模型提供了统一的建模框架
  4. 预测应用：ARMA模型可用于短期收益率预测、风险价值(VaR)计算和期权定价
  5. 扩展：在ARMA基础上引入GARCH结构（ARMA-GARCH），可同时建模条件均值和条件波动率
实践提示

在实际金融建模中，收益率序列往往需要使用ARMA(1,1)或低阶ARMA模型。例如，沪深300指数日收益率常被建模为ARMA(1,1)-GARCH(1,1)过程，其中ARMA部分捕捉条件均值的动态，GARCH部分捕捉波动率聚类特征。
