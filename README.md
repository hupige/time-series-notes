直接在浏览器中打开任意 `tsa-chX/tsa-chX.html` 文件即可查看。

## 内容特色

- **交互式图表**：每章配备 ECharts 数据可视化图表（ACF/PACF、趋势拟合、波动率聚类等）
- **流程图**：使用 Mermaid 绘制分析流程图和知识体系树
- **金融案例**：结合金融工程实际应用场景（股价预测、Pairs Trading、GARCH 波动率等）
- **响应式设计**：支持桌面端和移动端阅读

## 技术栈

- 原生 HTML5 + CSS3
- ECharts 5.x（数据可视化）
- Mermaid 10.x（流程图/思维导图）
- 无框架依赖，纯静态页面
