Chapter 04

非平稳序列的确定性分析

趋势分析、季节效应与经典分解方法：从Wold到X-11，掌握非平稳时间序列的确定性因素提取与建模框架。

4.1时间序列的分解

在实际金融与经济数据分析中，绝大多数原始时间序列都是非平稳的。直接对非平稳序列建模往往会导致伪回归（Spurious Regression）问题。因此，首要任务是将序列中的各类确定性成分分离出来，以便对剩余的平稳随机成分进行进一步建模。

4.1.1 Wold分解定理（回顾）

在第三章中，我们已经学习了Wold分解定理。该定理指出：任何协方差平稳的纯非确定性随机过程，都可以表示为一组白噪声序列的无穷阶线性组合：

Wold分解 xt = μ + ∑j=0∞ ψj ϵt-j,    ψ0 = 1,   ∑j=0∞ ψj2 < ∞

然而，Wold分解的适用范围仅限于平稳序列。对于含有确定性趋势或季节效应的非平稳序列，我们需要更一般的分解框架。

关键洞察

Wold分解处理的是平稳随机成分；对于现实中的GDP、股价指数、零售额等金融经济序列，必须先提取趋势与季节等确定性成分，剩余部分才可能满足Wold分解的条件。

4.1.2 Cramer分解定理

Cramer分解定理将Wold分解推广到非平稳情形，奠定了确定性分析的理论基础：

Cramer分解 任何时间序列 {xt} 都可以分解为 确定性成分 Dt 与 随机性成分 Rt 之和：

xt = Dt + Rt

其中，确定性成分 Dt 可由过去的观测精确预测，而随机性成分 Rt 服从Wold分解。进一步地，在实际应用中，确定性成分通常被细化为以下四个子成分：

经典分解式 xt = Tt + St + Ct + It
📈

Tt：趋势成分

序列在长期中呈现的持续上升、下降或平稳的系统性变化方向。

📅

St：季节成分

由季节性因素（如季度、月份）引起的固定周期波动，周期通常小于一年。

🔄

Ct：循环成分

周期大于一年的反复波动，如经济周期中的繁荣—衰退—萧条—复苏。

🔬

It：不规则成分

除上述三者外的随机扰动，通常假设为白噪声或平稳序列。

在金融数据分析中，由于循环成分 Ct 与趋势成分 Tt 往往难以清晰区分，且循环周期不固定，因此常将二者合并处理，简化为 趋势—季节—不规则 的三因素分解。

4.2确定性因素分解

根据各因素之间相互作用方式的不同，经典的确定性分解模型主要分为加法模型与乘法模型两种形式。

加法模型

当季节波动的幅度不随趋势水平的变化而改变时，适用加法模型：

Additive Model xt = Tt + St + It

在加法模型中，各成分以绝对量的形式叠加。季节效应 St 满足 ∑ St = 0（在一个完整周期内），不规则成分 It 的期望为 0。该模型适用于金融数据中波动幅度相对稳定的序列，如某些利率序列的短期分析。

乘法模型

当季节波动的幅度随趋势水平的提高而按比例扩大时，适用乘法模型：

Multiplicative Model xt = Tt × St × It

在乘法模型中，季节效应以相对比例的形式作用于序列。此时季节指数满足 ∏ St = 1 或 ∑ St = m（m为周期长度，采用中心化指数形式）。乘法模型在金融经济领域更为常见，例如零售销售额、旅游收入等序列，其季节性波动的绝对金额往往随经济总量的增长而扩大。

技巧：对数变换

对乘法模型两边取对数，可将乘法关系转化为加法关系：

ln(xt) = ln(Tt) + ln(St) + ln(It)

这一技巧在实证分析中非常实用，使得我们可用加法分解的工具处理原本符合乘法特征的序列。

特征加法模型乘法模型
数学形式xt = Tt + St + Itxt = Tt × St × It
季节波动幅度恒定不变与趋势水平成比例
季节指数约束∑St = 0∏St = 1
适用场景波动幅度稳定波动幅度随趋势增长
金融案例短期利率残差GDP、零售额、股价成交量

4.3趋势分析

趋势提取是确定性分析的核心环节。本节介绍两类经典方法：趋势拟合法（参数方法）与平滑法（非参数/半参数方法）。

4.3.1 趋势拟合法

趋势拟合法假设趋势成分可以用一个明确的数学函数来刻画，通过估计函数参数得到趋势序列 T̂t。

线性趋势（最小二乘法）

当序列呈现近似直线的长期走向时，可设定线性趋势模型：

Linear Trend Tt = a + b · t + εt

利用普通最小二乘法（OLS）估计参数 a 与 b：

OLS Estimators b̂ = ∑(t - t̄)(xt - x̄)
∑(t - t̄)2,    â = x̄ - b̂ · t̄

曲线趋势

当趋势呈现加速增长或减速增长特征时，需要引入非线性函数形式：

二次趋势

Tt = a + bt + ct2

适用于存在单一拐点（加速或减速）的经济增长序列。

指数趋势

Tt = a · ebt

适用于以固定增长率持续扩张的序列，如复利增长的投资收益。

模拟数据：黑色散点为原始序列，蓝色实线为线性OLS拟合，绿色虚线为指数拟合。指数模型在后期更好地捕捉了加速增长特征。

4.3.2 平滑法

平滑法不预设趋势的具体函数形式，而是通过局部加权平均的方式提取趋势，具有更强的灵活性。

移动平均法（Moving Average）

对相邻的 k 期观测值取算术平均，以消除短期不规则波动：

Simple MA T̄t = 1
k ∑i=-(k-1)/2(k-1)/2 xt+i ,    k 为奇数

在金融市场中，k期均线（如5日均线、20日均线、60日均线）是移动平均法最直接的应用。短期均线对价格变化反应灵敏，长期均线则更好地反映中长期趋势方向。

指数平滑法（Exponential Smoothing）

指数平滑法对近期的观测值赋予更大的权重，权重按指数速度递减：

Single Exp. Smoothing T̄t = α xt + (1-α) T̄t-1 = ∑j=0∞ α(1-α)j xt-j

其中平滑参数 α ∈ (0,1) 控制衰减速度。α 越接近 1，对近期变化越敏感；α 越接近 0，平滑效果越强。

Holt线性趋势法

对于含有线性趋势的序列，一次指数平滑会产生滞后偏差。Holt法引入趋势项进行修正：

Holt's Method 水平项：ℓt = α xt + (1-α)(ℓt-1 + bt-1)
趋势项：bt = β(ℓt - ℓt-1) + (1-β) bt-1
预测：T̄t+h = ℓt + bt · h
金融案例：股价均线分析

在技术分析中，金叉（短期均线上穿长期均线）与死叉（短期均线下穿长期均线）被广泛用于判断趋势转折。例如，当5日均线上穿20日均线时，被视为短期趋势转强的信号；反之则为走弱信号。但需注意，均线策略在震荡行情中容易产生虚假信号。

趋势提取的建模步骤

  1. 序列可视化与初步判断

    绘制时序图，观察是否存在明显的长期走向（上升/下降/水平），判断趋势形态（线性/非线性）。

  2. 选择趋势提取方法

    若趋势形态清晰且稳定，选用参数拟合法（OLS）；若趋势形态复杂或可能变化，选用平滑法（移动平均或指数平滑）。

  3. 参数估计与模型拟合

    对拟合法估计回归系数；对平滑法通过交叉验证或信息准则选择最优平滑参数。

  4. 提取趋势并计算残差

    获得趋势估计值 T̂t 后，计算残差序列 et = xt - T̂t（加法模型）或 et = xt / T̂t（乘法模型）。

  5. 残差诊断

    检验残差序列是否平稳、是否存在自相关、是否仍含季节成分，以判断趋势提取是否充分。

注意：过度拟合风险

高阶多项式拟合或过小窗口的移动平均可能将随机噪声误认为趋势，导致过度拟合。在实际金融建模中，建议结合业务逻辑（如经济增长理论）与统计检验（如ADF检验）综合判断趋势提取的效果。

4.4季节效应分析

季节效应是金融经济时间序列中最常见的确定性成分之一。许多经济活动受自然条件（如气候）、制度安排（如节假日、财政年度）或社会习惯（如消费习惯）的影响，呈现出固定的年内周期性波动。

季节指数的概念与计算

季节指数（Seasonal Index）用于量化各时期相对于全年平均水平的波动程度。以月度数据为例，季节指数反映了该月份相对于年平均的典型偏离幅度。

计算步骤（以加法模型为例）：

  1. 计算历年同月的平均值 Ai（i = 1,...,12）；
  2. 计算所有月份的总平均值 Ā；
  3. 季节指数 Si = Ai - Ā（加法）或 Si = Ai / Ā（乘法）。

对于乘法模型的中心化季节指数，还需进行正规化处理，确保 ∏Si = 1 或 ∑Si = 12。

季节调整方法

季节调整（Seasonal Adjustment）是指从原始序列中剔除季节效应，得到季节调整后的序列（SA序列），以便更清晰地观察趋势与循环变动。

Seasonal Adjustment 加法：xtSA = xt - St = Tt + It
乘法：xtSA = xt / St = Tt × It

季节调整后的序列消除了固定周期波动，使得不同月份之间的数据具有可比性，是宏观经济分析与金融决策的重要依据。

金融案例：零售销售额的季节波动

中国社会消费品零售总额呈现出显著的季节性：每年1-2月受春节消费拉动出现高峰，6月受电商促销活动（618）影响出现次高峰，而节后月份通常回落。进行季节调整后，才能准确判断消费的真实增长趋势，避免将季节性高峰误判为经济过热。

金融案例：旅游行业的季节性

航空客运量、酒店入住率、景区门票收入等旅游相关行业指标具有极强的季节性。以航空业为例，暑期（7-8月）与黄金周（春节、国庆）为传统旺季，航班客座率显著高于淡季。投资者在分析旅游类上市公司财报时，必须将同比（与上年同期比）优于环比（与上月比）作为核心比较基准，以剔除季节因素的干扰。

4.5X-11过程简介

X-11是由美国普查局（U.S. Census Bureau）开发的季节调整程序，最早于1965年发布，后经多次改进（X-11-ARIMA、X-12-ARIMA、X-13ARIMA-SEATS），至今仍是世界各国官方统计机构广泛使用的季节调整标准工具。

X-11的核心思想

X-11基于移动平均法的迭代分解，通过多轮反复提取与修正趋势和季节成分，逐步逼近最优分解结果。其核心流程可概括为：

  1. 第一步：初步趋势估计

    利用中心化移动平均从原始序列中提取初步趋势—循环估计 TC(1)。

  2. 第二步：初步季节估计

    计算剔除初步趋势后的序列 x / TC(1)（乘法模型），通过跨期平均提取初步季节成分 S(1)，并进行平滑修正。

  3. 第三步：初步调整与再估计

    用初步季节成分调整原始序列，得到季节调整后的序列，再从中提取更精确的趋势估计 TC(2)。

  4. 第四步：迭代精炼

    重复上述过程，对极端值（离群点）进行识别与修正，不断精炼季节成分与趋势成分，直至各成分达到稳定。

  5. 第五步：输出最终结果

    输出最终的季节调整序列、趋势—循环序列、季节因子序列及不规则成分序列，并提供各类诊断统计量。

X-11的广泛应用

美国劳工部的非农就业数据、中国国家统计局公布的规模以上工业增加值、欧盟的HICP通胀数据等，均采用X-11或其改进版本进行季节调整。金融市场中，交易员高度关注这些经季节调整后的经济指标，因为它们更能反映经济的真实运行状况。

X-11的局限性

X-11本质上是一种基于经验规则的算法，而非统计模型。它无法提供参数估计的标准误，也难以直接进行假设检验。此外，对于含有结构性断点或大幅异常值的序列（如2020年新冠疫情期间的多数经济指标），X-11的自动调整效果可能不理想，需要人工干预或结合ARIMA建模（X-13ARIMA-SEATS）进行预处理。

本章所介绍的确定性分析方法——趋势拟合、平滑处理、季节调整与X-11分解——为后续学习ARIMA模型、协整分析以及波动率模型奠定了重要的预处理基础。在实际金融工程应用中，准确提取确定性成分不仅能够避免伪回归，还能帮助分析师识别经济周期的拐点、评估政策的真实效果，以及构建更稳健的预测模型。
